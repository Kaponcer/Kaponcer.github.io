# 常用快捷鍵 #
| 快捷鍵          | 用途           |                                                 example |
| :-------------- | -------------- | ------------------------------------------------------: |
| tab             | 自動補全程式碼 | html  這裡直接按tab鍵就可以補全為 ``` <html> </html>``` |
| ctrl + /        | 選取範圍註解   |                                                         |
| shift + alt + F | 快速排版       |                                                         |
|                 |                |                                                         |
